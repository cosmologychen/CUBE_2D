## 旁路保存粒子逻辑

## 保存粒子前的修正演化

### 一二阶理论

> \[!info] 修改时间> 2026-04-08

设粒子从第 $i$ 步演化到第 $i+1$ 步：

- **$r$**：粒子相对于观察者的共动坐标（距离）。
- $\chi$：光子的共动距离。由于 $\chi$ 是红移 $z$ 的函数，因此 $\chi$ 也是时间 $t$ 的函数。
- **$v$**：粒子的运动速度。
- **$t$**：时间。
  **穿越判据：** 在 $t\_i \to t\_{i+1}$ 时间段内，若粒子穿越光锥，则需满足以下条件：$\begin{cases} r\_i < \chi\_i \ r\_{i+1} > \chi\_{i+1} \end{cases}$

#### 一阶近似（线性插值）

假设在步长内粒子距离 $r$ 和光锥共动距离 $\chi$ 均随时间线性变化：

- **粒子位置插值**：$r(t) = r\_i + \frac{r\_{i+1} - r\_i}{t\_{i+1} - t\_i}(t - t\_i)$
- **光锥距离插值**：$\chi(t) = \chi\_i + \frac{\chi\_{i+1} - \chi\_i}{t\_{i+1} - t\_i}(t - t\_i)$
  **求解过程：**
  令 $r(t) = \chi(t)$，设距离差 $d = \chi - r$。
  解得穿越时刻 $t$ 为：$t = \frac{d\_i t\_{i+1} - d\_{i+1} t\_i}{d\_i - d\_{i+1}}$
  将 $t$ 代入插值公式即可得到对应的坐标 $(r, v)$。
  $$r = r\_i + (r\_{i+1} - r\_i) \frac{d\_i}{d\_i - d\_{i+1}}$$

#### 二阶近似（迭代优化）

通过考虑加速度 $\mathbf{F}/m$ 和哈勃参数 $H(a)$ 的非线性演化，利用牛顿迭代法（Newton-Raphson）精确锁定跨越点。

##### 迭代初始化

- 令初始时刻 $t' = t$（由一阶近似给出）。
- 计算当前状态：$a' = a(t')$，$\chi' = \chi(a')$，$r' = r(t')$，$v' = v(t')$。

##### 迭代循环体

1. **计算修正步长 ($dt$)：**
   利用残差除以相对速度：$dt = \frac{\chi' - r'}{\frac{c}{a'} + v'}$
2. **更新物理状态（二阶精度）：**
   - **时间：** $t\_{new} = t' + dt$
   - **膨胀因子：** $a\_{new} = a' + \dot{a} dt$
   - **光锥距离：** $\chi'= \chi(a') \approx \chi\_i + \frac{c}{H(z)} dz  = \chi\_i - c \times a dt$
   - **粒子速度：** $v\_{new} = v' + \frac{F\_i}{m a'} dt$
   - **粒子位置：** $r\_{new} = r' + v' dt$
3. **收敛判据：**
   计算相对误差：$$\epsilon = \left| \frac{r\_{new} - \chi\_{new}}{\chi\_{new}} \right|$$
   - 若 $\epsilon < \delta$：**收敛**，输出当前 $r\_{new}, v\_{new}$。
   - 否则：令 $t' = t\_{new}$，并更新所有变量进入下一次循环。
